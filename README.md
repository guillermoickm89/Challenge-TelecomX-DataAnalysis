# 📱 Challenge Telecom X - Predicción de Churn

Este proyecto forma parte del Challenge de Ciencia de Datos de Alura. El objetivo es identificar los motivos de la evasión de clientes y construir un modelo predictivo.

## 🛠️ Tecnologías utilizadas:
* **Python** (Pandas, Numpy)
* **Visualización** (Matplotlib, Seaborn)
* **Machine Learning** (Scikit-Learn - Random Forest)

## 📊 Conclusiones principales:
* El tipo de contrato **Mes a Mes** es el mayor indicador de fuga.
* Los clientes con **cargos mensuales altos** tienen mayor probabilidad de irse.
